# Game of Life

## Setup

we are using `rbenv` for ruby versioning
```shell
brew install rbenv
```

make sure to install the correct `.ruby-version`
```shell
rbenv install <ruby_version>
```

we are using [bundler][1] for gems
```shell
gem install bundler
```

## Testing

we are using [rspec][2] for testing.
```shell
rspec
```

[1]: https://bundler.io/
[2]: http://rspec.info/
